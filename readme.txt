== Description ==

A plugin that creates shortcodes for use with any affiliate cloaking service or plugin. Formats affiliate links so they aren't indexed by the search engines, ensures referral data is passed, and easier to style via CSS.

My Affiliate Link is NOT a link management system. It works with your existing tool to better format your affiliate links. 

[Read Full Documentation](https://larryludwig.com/plugins/my-affiliate-link/?utm_source=plugin&utm_medium=organic&utm_campaign=page)

== Works With ==
Works with all of the major link cloaking services:

* Thirsty Affiliates (Free and Pro versions)
* Pretty Links (Free and Pro versions)
* ClickMeter
